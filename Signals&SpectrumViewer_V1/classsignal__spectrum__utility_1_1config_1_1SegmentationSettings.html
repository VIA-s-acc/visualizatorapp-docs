Detailed Description

Settings for segmentation defaults used by window panel.

Attributes: default_method: Default segmentation method. parameter_mode: Parameter mode (manual/auto). window_size: Default window size. overlap_ratio: Default overlap ratio. energy_threshold: Default energy threshold. min_len: Default minimum window length. variance_radius: Default variance radius. variance_top_k: Default top-k points for variance mode. changepoint_penalty: Default changepoint penalty. crosscorr_threshold: Default cross-correlation threshold. manual_mode_enabled: Whether manual mode toggle is enabled by default.
