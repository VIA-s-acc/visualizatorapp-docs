Signal & Spectrum Utility

Signal & Spectrum Utility is a desktop Python application for multi-sensor signal visualization, segmentation, spectral transforms, and range-based comparison metrics.

Main capabilities

  • Load multi-sensor CSV datasets with channel mapping
  • Visualize time-domain signals and frequency-domain spectra
  • Compute segmentation windows with multiple methods
  • Compare absolute and pairwise spectral metrics by selected ranges
  • Export spectrum/comparison/windows and save/load full session state

Architecture

  • UI and orchestration: src/signal_spectrum_utility/
  • Data generator CLI: src/generate_sensor_data/
  • Entry points:
    • main.py
    • generate_sensor_data.py

Documentation scope

Doxygen is configured to parse Python docstrings (PYTHON_DOCSTRING = YES) and build API/reference docs for all source modules under src/.
