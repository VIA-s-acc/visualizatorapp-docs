Detailed Description

Serializable application session state.

Attributes: loaded_folder: Last loaded folder path. selected_channels: Selected channels by sensor. render_order: Explicit render order keys (sensor:channel). windows: Current segmentation windows. ranges: Selected spectrum ranges. settings: Plain dictionary of current settings.
