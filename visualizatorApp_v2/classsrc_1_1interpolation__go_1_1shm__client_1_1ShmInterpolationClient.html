Detailed Description

Shared memory client for interpolation service.

Protocol:

  1. Python creates/opens shared memory file
  2. Python writes request data to mmap
  3. Python sends lightweight TCP signal to Go (just "data ready" + offset info)
  4. Go reads request from mmap (zero-copy)
  5. Go writes result to mmap (zero-copy)
  6. Go sends TCP signal back ("result ready")
  7. Python reads result from mmap (zero-copy)

Layout of shared memory: [Header 64 bytes] [Request data region] [Result data region]

Header format: magic(4) | version(1) | status(1) | request_size(4) | result_size(4) | result_offset(4) | grid_shape_len(1) | grid_shape(12) | padding(34)
