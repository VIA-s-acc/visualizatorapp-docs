Detailed Description

Shared memory (mmap) client for Go interpolation service.

Uses memory-mapped files for zero-copy data transfer between Python and Go. Fastest IPC method - avoids TCP overhead for large arrays. Cross-platform: Windows (Named File Mapping), Linux/macOS (mmap on tmpfs/temp).
