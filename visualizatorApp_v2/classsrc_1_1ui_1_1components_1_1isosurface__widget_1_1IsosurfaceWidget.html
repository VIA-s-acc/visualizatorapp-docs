Detailed Description

Widget for isosurface configuration and building.

Provides UI controls for:

  • Setting target value and tolerance
  • Quick percentile presets (Q25, Median, Q75)
  • Colormap and opacity
  • Building and clearing isosurface
  • Keyboard shortcuts for fine-tuning
