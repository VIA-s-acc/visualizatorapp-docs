Detailed Description

Matplotlib visualization backend.

Backend for 2D plotting and basic 3D visualization using Matplotlib. Suitable for static plots, publications, and environments without interactive display support.

Display isosurface.

Creates a 3D isosurface from volumetric grid data. Uses Marching Cubes algorithm (skimage) for isosurface extraction.

Parameters

  • color: Single color (default: None, use colormap)
  • colormap: Colormap name (default: from config)
  • level: Multiple levels as list (overrides single level)
