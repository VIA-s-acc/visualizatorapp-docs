Detailed Description

CDZ format exporter.

Exports data to CDZ format (ZIP archive with structured contents). The archive contains metadata, raw data, merged data, interpolation results, and configuration in separate directories.

Structure: file.cdz (ZIP) ├── metadata.json ├── raw/sensor_data.npz ├── merged/merged_data.npz ├── merged/strategy.json ├── interpolation/grid_data.npz ├── interpolation/params.json └── config/export_config.json
