Detailed Description

Wrapper for Go interpolation client with 4-level fallback.

Adapts GoInterpolationClient to match the IInterpolator protocol. Uses 4-level fallback chain for optimal performance:

  1. Shared memory (fastest, zero-copy)
  2. Binary TCP (fast, optimized buffers)
  3. HTTP (compatible, slowest)
